A comprehensive WooCommerce extension to simplify GST compliance and reporting for Indian e-commerce businesses.

== Description ==

**GST Audit Exporter** helps Indian WooCommerce store owners generate clean, compliant GST reports. Whether you're filing monthly, quarterly, or just need clean Excel output for your CA, this tool makes it easy.

**Features:**

- Export GST-ready Excel reports
- Detailed breakdown of CGST, SGST, IGST
- Assign and manage HSN codes to products
- Filter by order status, customer state, date range
- Works with all standard WooCommerce setups

Perfect for chartered accountants, finance teams, or small business owners managing GST compliance independently.

== Installation ==

1. Upload the plugin ZIP via **Plugins > Add New > Upload Plugin**.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **WooCommerce > GST Audit Exporter** to start using the plugin.

== Frequently Asked Questions ==

= Is this plugin compatible with the latest version of WooCommerce? =  
Yes, tested with WooCommerce 8.x and WordPress 6.5+.

= Can I export reports by specific date ranges? =  
Absolutely. Use the built-in filter to select custom date ranges.

= Where do I enter HSN codes? =  
Each product page will have a field under Product Data > General tab to enter the HSN code.

= Does it support IGST? =  
Yes. The plugin identifies whether CGST/SGST or IGST applies based on the customer's state.
